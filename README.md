# Three_Five

A simple python file that prints to the terminal a number, "Three", "Five" or "ThreeFive" for a given range.

## Instructions

Write a program that prints the numbers from 1 to 100.
But for multiples of three print "Three" instead of the number
and for the multiples of five print "Five".
For numbers which are multiples of both three and five print "ThreeFive".

### How to run the program

- clone the repository locally
- in the terminal from the project root directory enter the following command
  for Windows:

```
py main.py
```

for Mac or Linux

```
python3 main.py
```
